# Astro公式チュートリアル プロジェクト

## 概要
Astroの公式チュートリアルを学習しながら作成したサンプルプロジェクトです。  
静的サイトの構造やコンポーネントの使い方、ページルーティング、スロットなどの基本を学びました。

## 使用技術
- Astro
- HTML / CSS
- JavaScript（軽い動的要素）

## 学習内容・機能
- **ページルーティング**  
  Astroのページごとのルーティングの仕組みを理解
- **コンポーネントの作成と再利用**  
  ヘッダー、フッター、カードなどをコンポーネント化
- **スロット（`<slot />`）の使用**  
  レイアウトコンポーネントにページ固有の内容を差し込む方法を学習
- **軽量な動的要素**  
  フォームやボタンに簡単なJavaScriptを追加
- **基本的なCSSでのスタイリング**  
  レイアウトや色、フォントなどをCSSで調整
